# 대학 캠퍼스 내 분실물 찾기 플랫폼 

### **목적**: 
대학 캠퍼스 내에서 분실된 물건과 그 물건의 주인을 빠르게 찾을 수 있도록 도와주는 웹 플랫폼을 제공합니다.

### **기능**:
 - 분실물 등록
 - 분실물 검색
 - 분실물 알림
 - 분실물 주인 찾기
 - 분실물 반환 확인

#### **추가 고려사항**:
 - 보안
 - 사용자 인증
